# AlgoLatch v0.0.2-preview.2 for macOS and Windows

This prerelease provides the same current AlgoLatch source as two desktop
installers:

- `AlgoLatch-0.0.2-unsigned-universal.dmg` for Apple Silicon and Intel Macs;
- `AlgoLatch-Setup-0.0.2-unsigned-x64.exe` for Windows 10/11 x64.

Both packages are intended for early testing while trusted code-signing
credentials are unavailable. They are not the stable `v0.0.2` release.

## Important security notice

- The macOS application has a complete ad-hoc signature but is not signed with
  Apple Developer ID and is not notarized. Gatekeeper may require Control-click
  → Open or explicit approval in System Settings → Privacy & Security.
- The Windows installer is not Authenticode-signed. Windows can display an
  unknown-publisher or SmartScreen warning.
- Download only from this GitHub Release and verify `SHA256SUMS.txt` before
  opening either installer. Do not disable Gatekeeper or SmartScreen globally.
- This preview does not claim Apple, Microsoft, SignPath Foundation or any
  other trusted publisher signature.

## Installation

### macOS

1. Verify the DMG against `SHA256SUMS.txt` with
   `shasum -a 256 --check SHA256SUMS.txt`.
2. Open the DMG and drag **AlgoLatch** to **Applications**.
3. If Gatekeeper blocks the first launch, Control-click AlgoLatch in Finder,
   choose **Open**, and confirm the prompt.

### Windows

1. Verify the EXE against `SHA256SUMS.txt` with
   `Get-FileHash .\AlgoLatch-Setup-0.0.2-unsigned-x64.exe -Algorithm SHA256`.
2. Run the one-click per-user installer. Administrator access is not required.
3. Open AlgoLatch from the Start menu. The locked C toolchain is bundled; no
   Visual Studio, LLVM installation, gateway script or `PATH` change is needed.

Uninstalling either application does not delete managed projects stored in the
user's Documents folder.
